Welcome to Paparazzi

Looking for a gentle introduction to the Paparazzi autopilot? Use this one! If you have questions regarding paparazzi, please visit Gitter.im. If you have questions regarding this course, feel free to contact me. To use opencv: Make sure you have the latest version of the mavlabcourse branch from this remote: https://github.com/rmeertens/paparazzi navigate to your paparazzi folder and type: git submodule init && git submodule update go to paparazzi/sw/ext/opencv_bebop type make. This should take some time, and in the end you should have an install folder. Now you are ready to try the example cv_opencvdemo. In this example we convert the bebop image to grayscale, blur it and use a canny edge filter.
